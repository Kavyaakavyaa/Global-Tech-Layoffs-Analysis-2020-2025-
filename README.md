
## Overview

This project is an interactive Tableau dashboard that analyzes global technology layoffs from 2020–2025.

### Features

- 📈 Monthly layoff trend
- 🏢 Top 10 companies by layoffs
- 🌎 Global layoff distribution
- 🫧 Industry-wise layoffs
- 📊 KPI cards
- 🎯 Interactive filters

## Key Insights

- Over **581,000** employees were laid off.
- **1,327** companies were affected.
- The largest spike occurred during **2023**.
- Hardware and Consumer sectors experienced the highest layoffs.
